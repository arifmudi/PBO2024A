Q5. Jenis jenis tipe data 
    Tipe data numerik : 
    1. integer (int)
       ex : A = 3 -> tipe data angka bilangan asli
    2. Float (float) 
       ex : B = 3.5 -> tipe data angka bilangan bulat berkoma 

    Tipe data text : 
    1. string (str)
       ex : C = "aku" -> tipe data yang berbentuk teks
     
     Tipe data list 
     1. list (list)
        ex : D = ["tanah",3,"air"] -> tipe data utuk membuat list 
     2. tuple (tuple)
        ex : E = ("kaca","beling","roda") -> tipe data mirip list tapi menggunakan kurung biasa
     3. range (range)
        ex : F = range (5) -> untuk menentukan jumlah data, berguna untuk for 

     Tipe data boolean (bool)
        ex : G = true -> tipe data untuk menentukan true / false
